Node Types:
	1. normal - A normal node that has n options
	2. user_input - A node that requires some sort of input
	3. select_one - A user must select one option from a list of options
	4. terminal - This type of node represents the end of sequence. A user cannot continue past a terminal node

File structure
{
	"content" : [],			// I'm thinking that if we seperate the content from the node it may be easier to pictures and random strings and stuff
	"nodes" : []			// An array of nodes, specified below
}


Node structure
{
	"id" : 1,						// Must be an integer greater than 0
	"type" : "normal",				// One of the types specified above"
	"content" : "Hello World"		// The content to display to the user
	"options" : [],					// Array of options, specified below
	"choices" : ["Bob", "John"]		// Array of Strings specifying the choices to present to the user on a select_one node. This field is optional
}


Option structure
{
	"content" : "My name is...",	// Content to display to user
	"action" : "",					// Idk. May be useful later
	"visible" : true,				// Wether to display this option
	"default" : true,				// If true this is the default option
	"next" : 2						// The id of the next node if this option is selected
}
